Sistema de Reservas de Mesa - P3
Projeto desenvolvido para a disciplina de Desenvolvimento Web III. O sistema gerencia reservas de restaurante com um mapa visual interativo.

Autores
Anderson Fontes

Gustavo Silva dos Reis

Como Rodar o Projeto
Siga os 3 passos abaixo:

Passo 1: Iniciar o Backend
Abra o terminal na pasta backend.

Instale as dependências com o comando: npm install

Inicie o servidor com o comando: npm start

(Aguarde aparecer a mensagem: "MongoDB Conectado" e "Server rodando na porta 3000")

Passo 2: Abrir o Site
Vá até a pasta frontend.

Dê dois cliques no arquivo index.html.

Pronto! O sistema vai abrir no seu navegador.

Funcionalidades
Mapa Visual: Mostra mesas Livres (Verde), Reservadas (Amarelo) e Ocupadas (Vermelho).

Adicionar Mesa: Botão para criar novas mesas no mapa.

Reservar: Clique em uma mesa livre para reservar.

Editar/Cancelar: Clique em uma mesa ocupada para ver detalhes, editar ou cancelar.

Regras Importantes
Reservas precisam de 1 hora de antecedência.

Não é possível reservar se a mesa já estiver ocupada no horário.

O número de pessoas não pode ser maior que a capacidade da mesa.
